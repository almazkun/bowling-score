# bowling-score
Bowling score function

# Usage
```sh
git clone git@github.com:almazkun/bowling-score.git
cd bowling-score
python3
```
```py
from bowling_score import bowling_score
bowling_score([[10], [10], [10], [10], [10], [10], [10], [10], [10], [10, 10, 10]])
>>> 300
```
